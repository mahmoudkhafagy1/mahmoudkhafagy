## 🎯 About This Project

A **cyberpunk-themed personal portfolio** designed for an **Artificial Intelligence student**. Features stunning Matrix-style animations, floating code snippets, and automatic GitHub project integration. Every visual element is carefully crafted to reflect the world of **programming and AI**.

### ✨ Why This Portfolio?

- 🤖 **AI-Themed Design** - Every element relates to programming and artificial intelligence
- 💻 **Matrix Effect** - Iconic falling code animation in the background
- 🔄 **Auto GitHub Integration** - Projects are fetched automatically from GitHub API
- 📱 **Fully Responsive** - Works perfectly on all devices
- ⚡ **Lightning Fast** - Built with Vite for optimal performance

## 🎨 Features

### 🖥️ Background Animations (4 Layers!)
| Effect | Description |
|--------|-------------|
| 🟢 Matrix Rain | Falling Japanese characters and code symbols |
| 💾 Binary Rain | Cascading 0s and 1s with gradient colors |
| 💻 Code Snippets | Real programming code floating upward |
| 🔲 Digital Grid | Animated grid with pulsing intersection points |

### 🎮 Interactive Elements
- ✅ **Glitch Text Effect** on the name (cyberpunk style)
- ✅ **Terminal Box** showing real command-line output
- ✅ **Typing Animation** cycling through programming languages
- ✅ **Animated Progress Bars** for language proficiency
- ✅ **Scan Lines** on project cards (futuristic effect)
- ✅ **Rotating Rings** around profile image
- ✅ **Floating Particles** throughout the page
- ✅ **HTML Tag Headers** (`<About Me />`, `<Skills />`, etc.)

### 📄 Sections
1. **🏠 Hero** - Profile image, name with glitch effect, typing animation, terminal box
2. **👨‍💻 About** - Education, background, and highlights
3. **🛠️ Skills** - Categorized by Frontend, Backend, AI/ML, and Tools
4. **📂 Projects** - Auto-fetched from GitHub with smart image selection
5. **📊 Languages** - Animated progress bars showing proficiency
6. **📬 Contact** - Form with validation + social links

## 🚀 Getting Started

### Prerequisites
- Node.js (v16 or higher)
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/mahmoudkhafagy1/portfolio.git

# Navigate to project
cd portfolio

# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```

## 🌟 How GitHub Integration Works

1. 📡 Fetches repositories from your GitHub account via API
2. 🔍 Filters out forked repositories automatically
3. ⭐ Sorts by star count (most popular first)
4. 🖼️ Selects smart images based on language/topics
5. 📝 Displays description, language, topics, and stars
6. 🔗 Links directly to GitHub repo and live demo

### To add new projects:
1. Push your project to GitHub
2. Add a good description to the repository
3. Add topics (e.g., `react`, `python`, `ai`)
4. **That's it!** It appears automatically on your portfolio
